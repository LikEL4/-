1.  leeshop.sql을 데이터베이스로 가져옵니다.

2.  온라인 의약품 구매 시스템.zip을 IntelliJ IDEA로 가져옵니다.

3.  온라인에서 Lessshop>src>main>java>com.shop.bookshop>BookshopApplication을 찾습니다.BookshopApplication을 마우스 오른쪽 버튼으로 클릭하고 'BookshopApplication' 실행을 선택하여 프로그램을 실행합니다.

4.  새 Google 페이지를 열고 http://localhost:8088을 입력하여 온라인 의약품 구매 시스템의 홈페이지에 로그인합니다.

5.  오른쪽 상단에서 로그인(id:abc, 비밀번호:123456) 또는 등록을 선택할 수 있습니다. 검색창에서 의약품을 검색할 수 있습니다. 다른 카테고리를 클릭하면 분류된 의약품을 볼 수 있습니다. 의약품을 클릭합니다. 약의 상세 정보를 보려면 장바구니에 담기를 선택하거나 약을 직접 구매하려면 약 구매 시 주문 정보를 입력하여 주문을 제출합니다.

6.  로그인 페이지 우측 하단의 백그라운드 로그인을 클릭하고 계정 비밀번호(id: admin, 비밀번호: 123456)를 입력하여 관리자 인터페이스로 진입하여 약품 정보, 주문 정보, 소비자 정보 등을 관리할 수 있습니다.
